## 📅 Sales Process Flow

### Stage 1: Initial Outreach
**Goal:** Get 20-minute demo meeting

**Materials:**
- Email template (short version)
- One-Pager attachment

**Actions:**
1. Send personalized email
2. Follow up 2-3 times if no response
3. LinkedIn connection/message
4. Phone call if email fails

---

### Stage 2: Demo Meeting
**Goal:** Demonstrate value, schedule technical deep-dive

**Materials:**
- Live demo (use demo.html)
- Energy Sector Brief
- ROI Calculator (pre-filled with estimates)

**Actions:**
1. 20-minute live demo (Vibe Coding + Agentic AI)
2. Discuss specific use cases
3. Show ROI calculator with their estimated numbers
4. Schedule technical deep-dive with their team

---

### Stage 3: Technical Deep-Dive
**Goal:** Validate technical fit, agree on pilot scope

**Materials:**
- Detailed technical architecture
- Integration protocol documentation
- Security and compliance documentation
- Reference customer contacts

**Actions:**
1. Technical Q&A with their IT team
2. Review integration landscape
3. Identify pilot use case
4. Address security/compliance concerns

---

### Stage 4: Pilot Proposal
**Goal:** Get pilot approval and contract signature

**Materials:**
- Pilot Proposal (customized)
- ROI Calculator (refined with actual data)
- Contract/SOW

**Actions:**
1. Present customized pilot proposal
2. Review ROI analysis
3. Negotiate terms if needed
4. Execute contract

---

### Stage 5: Pilot Execution
**Goal:** Deliver successful pilot, prove ROI

**Materials:**
- Project plan
- Weekly status reports
- Final presentation deck

**Actions:**
1. Execute 30-day pilot per proposal
2. Weekly stakeholder updates
3. Final presentation with results
4. Proposal for full deployment
